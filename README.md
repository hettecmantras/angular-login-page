# Angular Login Page

This repository contains a minimal Angular single-page application that renders a responsive login card backed by a reactive form and a stubbed authentication service.

## Requirements

- Node.js 18+
- npm 10+

## Getting Started

```bash
cd frontend
npm install
npm start
```

This will host the app locally at http://localhost:4200.

## Building & Testing

- `npm run build` – build the production bundle
- `npm test` – run the Angular unit tests (ChromeHeadless)
- `npm run lint` – run the linter (deprecated TSLint for legacy compatibility)
